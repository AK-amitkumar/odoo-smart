EXPENSE REPORT          
Start here by writing the project name, your name and today's date.

Fill in the grey marked fields. Collect receipts for a project or time period.

This template has five sheeets that you navigate between in the bottom of this document 
RECEIPTS                      
Numerate your receipts and write the information into the correct row. Write what you bought and why under specification.  
Write total amount including VAT and choose the correct VAT rate for each receipt.
Which currency? The report is made in Swedish krona. If you have expenses in an other currency then use www.riksbank.se to find the exchange rate.
PER DIEM                      
Write the destination of your trip, if international, choose the country. Write departure and return date, and if the departure began 
before or after 12.00 and if you arrived before or after 19.00. You can report up to 4 travels in the template.
VEHICLE REIMBUSRMENT                  
Write vehicle type and registration number of the vechile. Then you write departure and return date, where you traveled, 
total amount of km and purpose of the travel. You can also write potential congestion charges that you have payed in connection to travelling 
(in that case a copy of the invoice from the Swedish Transport Agency has to be sent to SMartSe.)
